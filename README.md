# Aquarium AI Monitoring System

Prototype system for monitoring aquarium fish using computer vision.

## Features
- Live aquarium camera feed
- AI fish detection
- Fish count tracking
- Dashboard monitoring

## Tech Stack
- FastAPI
- React
- Roboflow AI
- SQL Server
- Raspberry Pi + Arducam

## Run Backend

uvicorn backend.main:app --reload

## Run Frontend

cd aquarium-dashboard  
npm start
